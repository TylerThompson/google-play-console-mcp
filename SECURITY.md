# Security Policy

Thanks for helping keep `google-play-console-mcp` and its users safe.

## Supported Versions

This is an open-source project. Security fixes are applied to the latest code on the default branch.

| Version | Supported |
| ------- | --------- |
| Latest (`main`) | :white_check_mark: |
| Older releases/commits | :x: |

## Reporting a Vulnerability

Please do **not** open public GitHub issues for suspected security vulnerabilities.

Instead, use one of these private channels:

- **Preferred:** [GitHub Security Advisories](https://github.com/tylerthompson/google-play-console-mcp/security/advisories/new)
- **Email:** [security@tylerthompson.me](mailto:security@tylerthompson.me)

If possible, include:

- A clear description of the issue and impacted components
- Steps to reproduce or a proof of concept
- Potential impact/severity
- Suggested remediation (if known)

## What to Expect

- Initial triage response target: **within 5 business days**
- Status updates during investigation: **as available**, usually within **7 business days**
- If accepted, a fix will be prepared and released as soon as practical
- If declined, you will receive an explanation

## Disclosure Policy

Please allow time for investigation and remediation before public disclosure.  
After a fix is available, we encourage coordinated disclosure and proper credit to reporters who wish to be acknowledged.

## Scope

Security concerns in scope include vulnerabilities in this repository's source code, release artifacts, and documented setup instructions.

Out of scope:

- Third-party service outages or upstream provider issues not caused by this project
- Vulnerabilities requiring physical device compromise or local administrator compromise

## Safe Harbor

We appreciate good-faith security research.  
If you act in good faith, avoid privacy violations/data destruction/service disruption, and give us a reasonable time to resolve issues before disclosure, we will not pursue legal action for your research.
